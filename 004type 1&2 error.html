Statistical Hypothesis Errors: Type I and Type II

Note: The terms Type 1 Error and Type I Error (using Roman numeral I) are identical and interchangeable.

Type I Error (False Positive) üö®

A Type I Error occurs when you reject the null hypothesis (H0) when it is actually true.

  • Name: False Positive
  • Analogy: A medical test says you have a disease (positive result), but you are actually healthy (false alarm).
  • In Statistics: You conclude there is a significant effect or difference when, in reality, there is not.
  • Symbol: The probability of a Type I Error is denoted by alpha (α), also known as the significance level.

Type II Error (False Negative) ‚ö†Ô∏è

A Type II Error occurs when you fail to reject the null hypothesis (H0) when it is actually false.

(This is the error that represents a "false negative" and is included for comparison.)

  • Name: False Negative
  • Analogy: A medical test says you are healthy (negative result), but you actually have the disease (missed case).
  • In Statistics: You conclude there is no significant effect or difference when, in reality, there is one.
  • Symbol: The probability of a Type II Error is denoted by beta (β).
